# Список задач (Todo List)
___
## Описание

Приложение "Список задач" позволяет удобно управлять вашими задачами. Вы можете добавлять, редактировать и удалять задачи, а также отмечать их как выполненные.
___
## Особенности
- Отображение задач
В планах:
- Добавление новых задач
- Редактирование существующих задач
- Удаление задач
- Отметка задач как выполненные
- Фильтрация задач по статусу (все, активные, выполненные)
- RESTful API для взаимодействия с фронтендом
___
## Технологии

- Java
- Spring Boot
- БД(в планах)

## Установка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/lloreley/ToDoList.git

2. Перейдите в директорию проекта:
   ```bash
   cd todo-list
   
3. Запустите приложение:
   ```bash
   ./mvnw spring-boot:run
   
4. Откройте браузер и перейдите по адресу: http://localhost:8080
